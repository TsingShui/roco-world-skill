# 洛克王国 游戏规则

## 数据资产

Skill 目录下包含完整游戏数据，需要时按路径读取：

- `skills/roco-world/skills.csv` — 487 条技能数据（字段：name, element, category, cost, power, effect）
- `skills/roco-world/pets/<name>.json` — 606 个宠物详情（文件名含 `/` 的已编码为 `%2F`）

## 对战概述

回合制对战，整体风格类似宝可梦：双方各出一只精灵，每回合同时选择行动（使用技能或切换精灵），按速度/先手决定出手顺序。

**胜利条件：** 每位玩家有 4 格生命值，击败对方一只精灵扣对方 1 格并恢复自己 1 格，一方生命值归 0 则判负。

## 属性系统

游戏共有 19 种属性：普通、草、火、水、光、地、冰、龙、电、毒、虫、武、翼、萌、幽、恶、机械、幻、无

每种属性有四个维度，各分两档：

| 维度 | 普通 | 强力（触发条件） |
|------|------|------|
| strong（攻击克制） | 200% | 300%（目标的两个属性均被攻击属性克制） |
| resist（攻击被抵抗） | 50% | 25%（目标的两个属性均抵抗攻击属性） |
| weak（防御弱点） | 200% | 300%（自身两个属性均被攻击属性克制） |
| vulnerable（防御抵抗） | 50% | 25%（自身两个属性均抵抗攻击属性） |

## 能量系统

- 能量属于宠物个体，换场后保留
- 首次出场获得 10 点能量
- 使用恢复技能回复 5 点能量
- 每回合只能使用一个技能（消耗对应能量），切换精灵也算消耗一个回合行动

## 词条系统

### 印记

常驻在战场的效果。每一方同时只能存在 1 种正面印记和 1 种负面印记。

| 印记 | 效果 |
|------|------|
| 减速 | 速度 -10 |
| 光合 | 回合结束时，每回合每层恢复 1 点能量 |
| 攻击 | 每层使技能威力 +10% |
| 蓄势 | 全攻击技能威力 +30%，且能耗 +1 |
| 蓄电 | 入场首回合技能威力 +10 |
| 湿润 | 每层湿润印记使所有技能能耗 -1 |
| 降灵 | 精灵入场时（所有情况），每层失去 1 能量 |
| 龙噬 | 释放 3 能耗技能后，获得双攻 +30% |
| 风起 | 每层先手攻击时，使伤害提升 20%；仅生效 1 层 |
| 棘刺 | 精灵入场时（所有情况），每层造成 6% 最大生命值伤害 |
| 星陨 | 使用非幻系技能攻击拥有此印记的精灵时，触发星陨印记，消耗全部星陨层数，对其造成额外的幻系伤害 |
| 中毒 | 中毒效果。回合结束时，造成 3%（向下取整）毒属性伤害，受到属性克制影响 |

### 负面

| 词条 | 效果 |
|------|------|
| 寄生 | 回合结束时，每层吸取 6% 最大生命值，换下场后清除；草系精灵免疫 |
| 冻结 | 每层冻结目标 5% 的最大生命值，冻结部分的生命值不可用，换下场依然存在；冰系精灵免疫 |
| 中毒 | 回合结束时，造成 3%（向下取整）毒属性伤害，受到属性克制影响，换下场后清除；毒系精灵免疫 |
| 灼烧 | 回合结束时，造成 2% 火属性伤害，受到属性克制影响，并衰减一半（向上取整）的层数，最少衰减 1 层，换下场后清除；火系精灵免疫 |
| 萌化 | 变为进化链的上一阶形态，种族值同步降低（二阶 80%，一阶 60%）；最低形态时无法继续萌化；需要萌化获得增益的技能在最低形态时不会生效；换下场依然存在 |

### 天气

对双方精灵均生效的场地效果，持续 8 回合，可被顶替。

| 天气 | 效果 |
|------|------|
| 下雨 | 水系环境，水系技能提升 50% |
| 沙暴 | 地系技能能耗减半 |
| 暴风雪 | 每回合赋予在场精灵 2 层冰冻效果 |

### 其他关键词

| 关键词 | 效果 |
|--------|------|
| X号位 | 从上到下第 X 个位置的技能 |
| 进发 | 进入战场首次行动获得额外效果 |
| 连击 | 发起 X 次攻击，每次攻击威力为技能威力（X 为连击数） |
| 先手+X | X 更高的一方优先出手，X 相同时速度更快的一方先出手；该效果不可叠加 |
| 折返 | 释放后精灵返回背包，需换另一只精灵入场；入场首回合免疫此效果 |
| 脱离 | 与折返相同 |
| 紧急脱离 | 精灵离场，并随机选择队伍里一只其他精灵替换入场；入场首回合免疫此效果 |
| 返场 | 回到背包，并再次入场；入场首回合免疫此效果 |
| 蓄力 | 本回合蓄力并扣除能量，下回合只可释放已蓄力技能、聚能或下场；蓄力期间免疫所有离场效果 |
| 积蓄 | 状态技能。可继续蓄力一回合，最终技能可额外释放次数 +1；被打断时不会消除蓄力 |
| 应对 | 若敌方本回合使用的技能为我方应对的技能类型，则应对成功，立即释放技能 |
| 迅捷 | 主动更换精灵入场时，按技能排序使用第一个能量满足要求且带有迅捷的技能 |
| 奉献 | 使技能「啃咬」和「虫群」获得额外强化，全部精灵共享，效果可叠加：威力+20、攻击附加 2 层中毒、20% 吸血、连击数+1、能耗-2 |

## 首领形态 / 愿力冲击

每场战斗二选一，只能触发其中一种：

- **首领形态**：类似宝可梦 Mega 进化。特定精灵触发后形态改变、属性大幅提升，持续整场。
- **愿力冲击**：类似宝可梦 Z 招式。每场一次性释放一个以精灵血脉属性为属性的高威力技能。

## 性格系统

性格决定精灵的属性成长方向：提升一项属性 10%，同时降低另一项属性 10%。共 25 种性格。

后期装备、觉醒、天赋加成均为乘法叠加，10% 差距会持续放大，极品与废品性格最终攻击力可差 20%-30%。
